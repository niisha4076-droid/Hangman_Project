🎮 Hangman Game (OOP in Python)

📌 ***Description***

This project is a console-based Hangman game developed using Object-Oriented Programming (OOP) concepts in Python.
The game allows a player to guess a hidden word one letter at a time with limited attempts.

The program provides hints to help the user and visually displays the hangman drawing after each incorrect guess.

---

🎯 **Features**

- Word guessing game with limited attempts
- Multiple hints for each word
- Progressive hint system (hints appear after wrong guesses)
- ASCII-based Hangman drawing
- Tracks correct and incorrect guesses
- Prevents duplicate guesses

---

🧠 ***OOP Concepts Used***

- Class → Hangman
- Object → Instance of the Hangman class
- Encapsulation → Data and methods inside the class
- Methods → display_word(), guess_letter(), play(), etc.

---

🛠️ ***Technologies Used***

- Python 3
- Random module

---

▶️ **How to Run**

1. Install Python on your system
2. Save the file as "hangman.py"
3. Open terminal or command prompt
4. Run the program using:
   python hangman.py

---

🎮 **How to Play**

- The system selects a random word
- Player guesses one letter at a time
- Correct guesses reveal letters
- Wrong guesses reduce attempts and draw hangman
- Game ends when:
  - Player guesses the word (Win) 🎉
  - Attempts reach zero (Lose) 💀
